README :
-----------
In order to create groundtruth just select your parameter in the create_gt.py file 

then run the file
the created npz needs to be placed in the comparison folder
you can now have it to compare with your nosetests or by running directly the comparison file

to look into the .npz, use the see() function of comparison.py

Only a file named groundtruth.npz will be considered as the gt.
